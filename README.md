# 📦 Inventory Optimization Project

## 📌 Overview

This project analyzes inventory data using Python to optimize stock levels, identify fast- and slow-moving products, and determine when to reorder items.

## 🎯 Objectives

* Calculate inventory turnover
* Identify products that need restocking
* Analyze product performance
* Provide data-driven insights for inventory management

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## 📊 Key Analysis

### 1. Inventory Turnover

* Calculated how many days it takes to sell inventory
* Identified fast vs slow-moving products

### 2. Reorder Point

* Formula: daily_sales × lead_time_days
* Determined when stock should be replenished

### 3. Stock Risk Analysis

* Flagged products at risk of stockout
* Helped prioritize restocking decisions

## 📈 Results

* Identified top fast-selling products
* Highlighted slow-moving inventory
* Provided actionable restocking recommendations

## 📁 Project Structure

* inventory_data_clean.csv → dataset
* Inventory_Optimization.ipynb → analysis code

## 🚀 How to Run

1. Download the dataset
2. Open the notebook in Jupyter
3. Run all cells

## 💡 Key Insights

* Products with low turnover days sell faster
* Some products are at risk of stockout
* Data can help improve inventory efficiency
